# Vidéos de démonstration — App ChatGPT

Dossier pour déposer les vidéos de démo de l'app « Assurance temporaire ».

## Fichiers
- `widget-demo.webm` — clip du widget de devis (durée → prix recalculé). Complément / B-roll.
- `demo-chatgpt.mp4` — **À AJOUTER** : ta vraie capture d'écran de l'app dans ChatGPT
  (mode développeur), pour le champ « Demo Recording URL » de la soumission OpenAI.
  Voir le script dans `../../docs/dossier-openai.md` / la conversation.

## Obtenir une URL publique pour le formulaire OpenAI
Le champ « Demo Recording URL » attend une **URL**. Deux options :
1. **YouTube (non répertorié) ou Loom** — recommandé (lecture en streaming fiable).
2. **Via ce repo** : après commit + push, l'URL « raw » du fichier peut servir :
   `https://raw.githubusercontent.com/Stachmou34/tempo-assurance/main/chatgpt-app/demo/demo-chatgpt.mp4`
   (à vérifier que le lecteur OpenAI accepte la lecture directe ; sinon, préférer l'option 1).

## Format
- Idéalement **.mp4**, < 2 min, sans musique.
- Doit montrer : mode développeur, l'appel des outils (devis), au moins 1–2 cas d'usage.
